School Projects

Worbi

A mobile application to learn English, built in Kotlin. It features a memory game, Tinder-like cards to review words, and pronunciation to train vocabulary.

It also includes functionality for reading books in English, allowing users to improve reading comprehension while expanding their vocabulary.

Additionally, it offers interactive lessons with multiple-choice options, where users can practice building correct sentences, reinforcing grammar and writing skills.

Adventure Time Online Game

A 2D online Adventure Time game created with JavaScript, HTML, and CSS. It has simple physics for jumping and attacks, basic sprite animations, two levels, and a boss with movement and animation phases, where penguins are generated as enemies. The boss changes behavior depending on remaining health.

The game has a static background, music, sound effects, and keyboard controls, along with a nice aesthetic reminiscent of the original series.

MemeHero

A Java video game aiming to be as similar as possible to Guitar Hero, including both videos and music in parody form, featuring uncommon characters. It has a very simple design and gameplay very similar to Guitar Hero.

It also includes a multiplayer mode where players can compete against each other, and an editor mode that allows modifying songs and the positioning of notes during the song.

Memory Extreem

It’s a memory card game with an option to load new words and show results in a local file. The game is designed to run in the terminal, so it can work on any operating system.

The project was developed in C++ and a bit of C, using Makefile for compilation and dependency management.
